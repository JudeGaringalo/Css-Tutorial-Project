# CodeSync CSS Tutorial Project

CodeSync CSS Tutorial Project is a static educational website created as a university project. It provides beginner-friendly CSS lessons, code examples, output previews, and exercises to help users understand the fundamentals of Cascading Style Sheets.

The website is designed as a simple learning platform where students can explore different CSS topics, review sample syntax, and answer practice questions.

## Project Description

This project introduces basic to intermediate CSS concepts through a structured tutorial website. It includes explanations, sample code snippets, visual outputs, and quiz-style exercises for learners who are starting to study web design and front-end development.

The website also includes information about Rizal Technological University, including its mission, vision, and philosophy, along with a developers section showing the project team.

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

Make sure you have the following installed:

- Git
- A web browser
- A code editor such as Visual Studio Code

### Installation

Clone the repository:

```bash
git clone https://github.com/JudeGaringalo/Css-Tutorial-Project.git
```

Go to the project folder:

```bash
cd Css-Tutorial-Project
```

Open the project in your code editor:

```bash
code .
```

Open `index.html` in your browser.

You can also use the Live Server extension in Visual Studio Code for a better local development experience.

## How to Use

1. Open the website.
2. Start from the Home section.
3. Go to CSS Topics to study the lessons.
4. Review the examples and output previews.
5. Go to CSS Exercises to answer practice questions.
6. Visit the Developers section to view the project team.
